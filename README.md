## 🚀 DSA Playground 

A C++ based Data Structures & Algorithms practice playground.

This repository contains implementations of core DSA concepts written from scratch
to strengthen fundamentals, problem-solving skills, and low-level understanding.

---

##  Topics Covered

###  Arrays
- Traversal
- Insertion & Deletion
- Linear Search
- Binary Search
- Max / Min
- Reverse Array
- Sorting (Bubble Sort)

---

###  Singly Linked List
- Insert at Head
- Insert at Tail
- Delete by Value
- Search
- Length
- Display
- Reverse Linked List

---

###  Doubly Linked List
- Insert at Head
- Insert at Tail
- Delete by Value
- Display Forward
- Display Backward
- Length

---

###  Stack (Array Implementation)
- Create Stack
- Push
- Pop
- Peek
- isEmpty
- Display
- Free Stack

---

###  Queue (Array Implementation)
- Create Queue
- Enqueue
- Dequeue
- Peek
- isEmpty
- isFull
- Display
- Free Queue

---

##  Tech Stack
- Language: C++
- Compiler: g++
- OS: Linux

---

##  How to Run
- Clone the repository
	- git clone https://github.com/abhishekmail-prog/DSA-Playground.git
	- cd DSA-Playground

- Compile and Run:
	- make run
